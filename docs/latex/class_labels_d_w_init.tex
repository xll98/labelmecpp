\doxysubsection{Detailed Description}
负责标签列表组件的初始化及状态管理 

Labels\+D\+W\+Init类负责标签列表浮动窗口组件的界面初始化， 以及后续的标签列表的管理。 由\+Main\+Window连接该类与\+Central\+Widget类的信号与槽， 建立起双向信息传递通道以进行同步。
