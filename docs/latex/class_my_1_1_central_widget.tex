\doxysubsection{Detailed Description}
中心组件基类 

Central\+Widget类负责主窗口中心组件的界面初始化以及后续中心组件的管理。 其两个派生类分别负责2\+D、3\+D中心组件， 2d和3d的模式可以进行切换。 内部的绘画以及标注相关的动作交由\+Label类（或\+Label3\+D类）进行处理。 命令的撤销和重做交由\+Command2\+D类（或\+Command3\+D）进行处理。 标注文件的解析和存储，以及3\+D图片系列文件的解析均交由\+Parse类进行处理。
